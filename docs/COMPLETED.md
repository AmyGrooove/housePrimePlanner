## Что работает прямо сейчас

### 1. Система Undo/Redo
```tsx
// Кнопки в правом верхнем углу
<UndoRedoControls canUndo={...} canRedo={...} onUndo={...} onRedo={...} />

// Горячие клавиши
Ctrl+Z - отменить
Ctrl+Y - повторить
Delete - удалить выбранное
Escape - отменить текущее действие
```

### 2. Валидация данных
- ❌ Нельзя создать стену короче 100 мм или длиннее 50 м
- ❌ Нельзя создать стену тоньше 50 мм или толще 1000 мм
- ❌ Нельзя создать пересекающиеся стены
- ❌ Нельзя сохранить объект с пустым названием
- ❌ Нельзя создать дубликат объекта
- ✅ Все ошибки показываются пользователю с понятными сообщениями

### 3. Площадь комнаты в реальном времени
При рисовании стен в RoomBoard автоматически отображается:
```
Площадь: 25.50 м²
```

- Мемоизированы тяжелые вычисления

### Отменить/Повторить действие
1. **Кнопками:** Нажмите кнопки в правом верхнем углу
2. **Клавиатурой:** Ctrl+Z для отмены, Ctrl+Y для повтора
